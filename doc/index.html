Change App Package Name -

To rename the package name in Flutter, you can use the change_app_package_name library. This library allows you to change the app package name with a single command, making the process quick and easy.


What This Library Does?
  • Updates AndroidManifest.xml files for release, debug, and profile
  • Updates the build.gradle file
  • Updates the MainActivity file (supports both Java and Kotlin)
  • Moves the MainActivity file to the new package directory structure
  • Deletes the old package name directory structure

Steps to Use This Library for Renaming a Package in Flutter

1. Add change_app_package_name to the dev_dependencies section of your pubspec.yaml file.

dev_dependencies:
	change_app_package_name : ^1.2.0

2. Update dependencies

flutter pub get

3. Run the following command to change the package name

flutter pub run change_app_package_name:main com.new.package.name

To change the app logo in Flutter, you can use the Flutter Launcher Icons library. This command-line tool makes it easy to update the launcher icon for your Flutter app.


Here are the steps to utilize this library for changing the logo in Flutter:

1. Setup the config file

Incorporate your Flutter Launcher Icons configuration into your pubspec.yaml or generate a new configuration file named flutter_launcher_icons.yaml. See the example below:

When naming your configuration file anything other than flutter_launcher_icons.yaml or pubspec.yaml, you must specify the file name when executing the package.

flutter pub get
flutter pub run flutter_launcher_icons -f <your config file name here>

2. Run the package

After setting up the configuration, all that is left to do is run the package.

flutter pub get
flutter pub run flutter_launcher_icons
